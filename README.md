## Requirements

- Node.js `lts/iron`
- pnpm `v10.x`

## Getting Started

1. Clone this repository

   ```bash
   git clone https://github.com/coderwaska/wastu-fyi.git
   ```

2. Move to the repository directory

   ```bash
   cd wastu-fyi
   ```

3. Install dependencies

   ```bash
   pnpm install
   ```

4. Run locally

   ```bash
   pnpm dev
   ```
